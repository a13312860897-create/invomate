# 📧 发票软件邮件配置完整指南

## 🎯 概述

本指南将帮助你完成发票软件的邮件配置，让你的系统能够自动发送发票、付款提醒等邮件。

## 📋 准备工作

在开始之前，你需要：
- ✅ 一个可用的邮箱账户
- ✅ 该邮箱的登录权限
- ✅ 10-15分钟的时间

---

## 🔧 第一步：选择邮件服务商

### 推荐方案对比

| 服务商 | 难度 | 稳定性 | 适用场景 | 推荐指数 |
|--------|------|--------|----------|----------|
| **Gmail** | ⭐ 简单 | ⭐⭐⭐⭐⭐ | 个人/小企业 | ⭐⭐⭐⭐⭐ |
| **Outlook** | ⭐⭐ 中等 | ⭐⭐⭐⭐ | 个人/企业 | ⭐⭐⭐⭐ |
| **企业邮箱** | ⭐⭐⭐ 复杂 | ⭐⭐⭐⭐⭐ | 企业用户 | ⭐⭐⭐ |

**💡 建议：如果你是初学者，强烈推荐使用 Gmail**

---

## 📧 第二步：Gmail配置（推荐）

### 2.1 开启两步验证

1. 登录你的 Gmail 账户
2. 点击右上角头像 → "管理您的Google账户"
3. 左侧菜单选择"安全性"
4. 找到"两步验证"并开启

### 2.2 生成应用专用密码

1. 在"安全性"页面，找到"应用专用密码"
2. 点击"应用专用密码"
3. 选择"邮件"和"Windows计算机"
4. 点击"生成"
5. **⚠️ 重要：复制生成的16位密码，这就是你的SMTP密码**

### 2.3 SMTP配置信息

```
SMTP服务器: smtp.gmail.com
端口: 587
加密方式: TLS
用户名: 你的Gmail邮箱地址
密码: 刚才生成的16位应用专用密码
```

---

## 📧 第三步：Outlook配置

### 3.1 SMTP配置信息

```
SMTP服务器: smtp-mail.outlook.com
端口: 587
加密方式: STARTTLS
用户名: 你的Outlook邮箱地址
密码: 你的Outlook密码
```

### 3.2 注意事项

- Outlook可能需要在账户设置中允许"不太安全的应用"
- 如果开启了两步验证，也需要生成应用密码

---

## 🔧 第四步：在系统中配置

### 4.1 访问配置页面

1. 打开你的发票软件
2. 访问：`http://localhost:3000/email-config-test`
3. 你会看到邮件配置测试界面

### 4.2 配置步骤

1. **检测邮箱服务商**
   - 在"邮箱地址"框中输入你的邮箱
   - 点击"检测服务商"
   - 系统会自动识别邮箱类型

2. **测试邮件配置**
   - 点击"测试邮件配置"
   - 查看测试结果

3. **发送测试邮件**
   - 输入一个接收测试邮件的邮箱地址
   - 点击"发送测试邮件"
   - 检查是否收到邮件

## ⚠️ 常见问题解决

### 问题1：显示"邮件传输器未初始化"

**原因：** 系统还没有配置SMTP设置

**解决方案：**
1. 需要在后端配置SMTP设置
2. 或者通过管理界面添加SMTP配置

### 问题2：Gmail显示"用户名或密码错误"

**原因：** 使用了普通密码而不是应用专用密码

**解决方案：**
1. 确保开启了两步验证
2. 生成并使用应用专用密码
3. 不要使用你的Gmail登录密码

### 问题3：邮件发送失败

**可能原因：**
- 网络连接问题
- SMTP设置错误
- 邮箱服务商限制

**解决步骤：**
1. 检查网络连接
2. 验证SMTP配置信息
3. 查看错误日志
4. 尝试其他邮箱服务商
