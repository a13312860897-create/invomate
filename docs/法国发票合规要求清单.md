# 法国发票合规要求清单

## 概述
本文档记录了法国发票的所有强制性合规要求，确保发票在法国法律下具有法律效力。

**重要更新**：根据2025年最新法规，法国将从2026年9月1日起实施强制性B2B电子发票制度。

## 2025-2027年电子发票改革要点

### 实施时间表
- **2026年9月1日**：大型企业和中型企业必须能够接收电子发票 <mcreference link="https://marosavat.com/e-invoicing-france/" index="1">1</mcreference>
- **2027年9月1日**：小型和微型企业纳入强制范围 <mcreference link="https://www.vatupdate.com/2025/10/24/french-e-invoicing-mandate-a-comprehensive-briefing/" index="3">3</mcreference>
- **2026年1月**：开始要求发票行项目详细数据 <mcreference link="https://sovos.com/vat/tax-rules/e-invoicing-france/" index="2">2</mcreference>

### 技术要求
- **认证平台**：必须通过认证的服务提供商(PDP)交换发票 <mcreference link="https://sovos.com/vat/tax-rules/e-invoicing-france/" index="2">2</mcreference>
- **支持格式**：Factur-X、UBL、CII等结构化格式 <mcreference link="https://www.fiscal-requirements.com/news/1009" index="4">4</mcreference>
- **PEPPOL网络**：法国税务局已成为PEPPOL France权威机构 <mcreference link="https://marosavat.com/e-invoicing-france/" index="1">1</mcreference>

## 强制性数据条目

### 销售方信息 (Seller Information)
- [ ] **公司名称** (Company Name) - 必须与企业注册名称完全一致
- [ ] **SIREN号码** (SIREN Number) - 9位数字，企业唯一标识 <mcreference link="https://taxology.co/what-vat-applies-in-france/" index="1">1</mcreference>
- [ ] **SIRET号码** (SIRET Number) - 14位数字（SIREN + 5位分支机构代码NIC） <mcreference link="https://taxology.co/what-vat-applies-in-france/" index="1">1</mcreference>
- [ ] **增值税号** (VAT Number) - FR + 2位控制数字 + SIREN号码 <mcreference link="https://www.eurofiscalis.com/en/vat-registration-in-france/" index="2">2</mcreference>
- [ ] **公司地址** (Company Address) - 完整注册地址
- [ ] **联系方式** (Contact Information) - 电话、邮箱等

### 购买方信息 (Buyer Information)
- [ ] **客户名称** (Client Name) - 个人姓名或公司名称
- [ ] **客户地址** (Client Address) - 完整账单地址
- [ ] **客户VAT号** (Client VAT Number) - 如适用（B2B交易）
- [ ] **客户SIREN/SIRET** - B2B交易中必须包含 <mcreference link="https://www.vatupdate.com/2025/10/24/french-e-invoicing-mandate-a-comprehensive-briefing/" index="3">3</mcreference>

### 发票基本信息 (Invoice Basic Information)
- [ ] **发票号码** (Invoice Number) - 唯一且连续的编号
- [ ] **发票日期** (Invoice Date) - 开具日期
- [ ] **服务/商品提供日期** (Service Date) - 实际提供服务或商品的日期
- [ ] **付款截止日期** (Payment Due Date) - 明确的付款期限

### 商品/服务明细 (Item Details)
- [ ] **商品/服务描述** (Description) - 清晰准确的描述
- [ ] **数量** (Quantity) - 销售数量
- [ ] **单价** (Unit Price) - 不含税单价
- [ ] **金额** (Amount) - 不含税金额
- [ ] **税率** (VAT Rate) - 适用的增值税率
- [ ] **操作类型** (Operation Type) - 商品、服务或混合类型 <mcreference link="https://sovos.com/vat/tax-rules/e-invoicing-france/" index="2">2</mcreference>
- [ ] **行项目详细数据** - 2026年1月起强制要求 <mcreference link="https://sovos.com/vat/tax-rules/e-invoicing-france/" index="2">2</mcreference>

### 财务信息 (Financial Information)
- [ ] **小计** (Subtotal) - 不含税总金额
- [ ] **增值税额** (VAT Amount) - 按税率计算的税额
- [ ] **总计** (Total) - 含税总金额
- [ ] **货币** (Currency) - 交易货币（EUR为默认）
- [ ] **VAT付款选项** - 必须明确标注VAT付款方式 <mcreference link="https://sovos.com/vat/tax-rules/e-invoicing-france/" index="2">2</mcreference>

## 增值税(VAT)规则

### 标准税率
- **20%** - 标准税率（大多数商品和服务）
- **10%** - 中间税率（餐饮、部分服务）
- **5.5%** - 降低税率（基本食品、书籍等）
- **2.1%** - 超低税率（药品、报纸等）

### 特殊规则
- [ ] **B2B交易** - 必须显示买卖双方的VAT号码
- [ ] **欧盟内交易** - 需遵守欧盟增值税规则
- [ ] **免税交易** - 需明确标注"免税"及原因

## 付款相关规则

### 付款信息
- [ ] **付款方式** (Payment Method) - 银行转账、支票等
- [ ] **银行信息** (Bank Details) - IBAN、BIC代码
- [ ] **预付款信息** (Advance Payment) - 如适用
- [ ] **滞纳金条款** (Late Payment Penalty) - 逾期付款条款

### 付款期限
- [ ] **法定期限** - 默认30天付款期限
- [ ] **延期付款** - 最长不超过60天
- [ ] **立即付款** - 可设置为立即到期

## 法律声明要求

### 必须包含的声明
- [ ] **服务提供者身份** - "服务提供者"标识
- [ ] **增值税豁免声明** - 如适用
- [ ] **合规保证声明** - 确保发票符合法国法律

### 可选但建议的声明
- [ ] **隐藏缺陷保证** (Garantie des vices cachés)
- [ ] **专业责任保险** - 如适用
- [ ] **知识产权保护** - 如适用

## 技术格式要求

### 电子发票
- [ ] **PDF格式** - 标准电子发票格式
- [ ] **数字签名** - 确保发票真实性
- [ ] **存档要求** - 保存期限10年
- [ ] **可读性** - 确保打印和数字阅读清晰

### 语言要求
- [ ] **法语** - 主要语言（必须）
- [ ] **双语** - 可添加英语等翻译
- [ ] **专业术语** - 使用标准商业术语

## 验证清单

### 开票前检查
- [ ] 所有必填字段完整
- [ ] SIREN/SIRET号码有效
- [ ] VAT计算正确
- [ ] 发票编号连续
- [ ] 日期逻辑正确

### 开票后检查
- [ ] 客户收到发票
- [ ] 发票已存档
- [ ] 会计记录更新
- [ ] 税务申报准备

## 2025年最新合规更新

### VAT注册门槛变化
- **服务业门槛**：从€37,500降至€25,000（2025年3月生效） <mcreference link="https://www.comarch.com/trade-and-services/data-management/e-invoicing/e-invoicing-in-france/" index="4">4</mcreference>
- **非欧盟企业**：必须在2025年12月31日前完成VAT注册 <mcreference link="https://www.vatupdate.com/2025/10/25/french-e-invoicing-mandate-a-comprehensive-briefing/" index="3">3</mcreference>

### 罚款标准更新
- **不合规罚款**：从€15提高至€50每张发票 <mcreference link="https://www.vatupdate.com/2025/10/24/france-proposes-2026-budget-law-amendments-to-e-invoicing-and-e-reporting-mandate/" index="5">5</mcreference>
- **未使用认证平台**：新增罚款，但有3个月纠正期 <mcreference link="https://www.vatupdate.com/2025/10/24/france-proposes-2026-budget-law-amendments-to-e-invoicing-and-e-reporting-mandate/" index="5">5</mcreference>
- **付款数据报告**：仅限于VAT按付款到期的服务交易 <mcreference link="https://www.vatupdate.com/2025/10/24/france-proposes-2026-budget-law-amendments-to-e-invoicing-and-e-reporting-mandate/" index="5">5</mcreference>

## 违规后果

### 法律风险
- **罚款** - 最高可达发票金额的5%
- **税务审计** - 可能触发税务稽查
- **法律效力** - 发票可能被认定为无效
- **商业影响** - 影响客户关系和信誉

### 合规建议
- 定期更新合规要求
- 使用专业发票软件
- 定期培训财务人员
- 保留所有合规记录
